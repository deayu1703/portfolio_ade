# Ade Ayu - Personal Portfolio

Selamat datang di repository portfolio pribadi saya.

Website ini dibuat untuk menampilkan informasi mengenai diri saya, pendidikan, kemampuan, sertifikat, serta menyediakan media komunikasi bagi recruiter maupun pihak yang ingin berkolaborasi.

## Fitur

-  Dark Mode
-  Responsive Design
-  Hero Section
-  Tentang Saya
-  Riwayat Pendidikan
-  Skills
-  Projects
-  Certificates
-  Download CV
-  Contact Form menggunakan EmailJS
-  Social Media Links

## Teknologi yang Digunakan

- HTML5
- Tailwind CSS v4
- Vanilla JavaScript
- EmailJS

## 📂 Struktur Project

## Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/USERNAME/portfolio_ade.git
```

2. Masuk ke folder project

```bash
cd portfolio_ade
```

3. Install dependency

```bash
npm install
```

4. Jalankan Tailwind CSS

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./dist/output.css --watch
```

5. Jalankan menggunakan Live Server.
